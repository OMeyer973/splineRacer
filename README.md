# splineRunner
IMAC semester 3 C++ and openGL 3 project. temple run clone but about a plane flying around a spline.

#### Install project
```shell
mkdir splineRacer-build
cd splineRacer-build
cmake ../SplineRacer
make
./PATH/TO/EXECUTABLE
```
